说明: 支持通过字符串、正则的方式匹配请求的URL， 且支持通过 !xxx 的方式取非，可以同时设置多个过滤条件， 条件之间用空格或换行符分隔，如： 
        #注释
        ke.qq.com !/test=1/ /token/i
        !m.ke.qq.com
上述配置表示，请求URL里面要包含字符串 ke.qq.com，或不匹配正则 /test=1/， 或匹配正则 /token/i，或不包含字符串 m.ke.qq.com，即： 
        if (url.indexOf('ke.qq.com') !== -1 || !/test=1/.test(url)
          || /token/i.test(url) || url.indexOf('m.ke.qq.com') === -1) {
          // auto save
        }
如果该插件无法满足需要，比如更多的过滤条件或存储到指定远程数据库，可以参考 whstle.token 的实现自定义一个whistle插件。
